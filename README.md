# Rednote Analytics Portfolio

A portfolio of end-to-end analytics and business intelligence projects built using Python, SQL Server, SSIS, SSAS, and SSRS.

This repository contains two connected analytics projects:

## 1. Rednote Restaurant Analytics

   - Python-based analytics pipeline

   - Data cleaning and feature engineering

   - Exploratory data analysis (EDA)

   - BI-ready dataset generation

## 2. IDS521 Rednote BI System

   - SQL Server data warehouse

   - SSIS ETL pipeline

   - SSAS OLAP cube

   - KPI modeling

   - SSRS business dashboards

---

# Portfolio Architecture


# Key Business Questions

This portfolio explores questions such as:

Which restaurant content characteristics generate stronger engagement?

Which cuisine categories deliver the highest engagement ROI?

Does premium pricing outperform budget positioning?

How does content structure affect user interaction?

Which posting schedules maximize engagement performance?

How can social content data be transformed into decision-ready business intelligence?

# Example Analytical Findings

Italian and Filipino cuisine categories generated higher engagement efficiency than dominant mainstream categories.

Medium-priced restaurants outperformed low-priced restaurants in weighted engagement.

Some niche content structures demonstrated stronger engagement than highly standardized formats.

Certain posting schedules generated consistently stronger engagement performance.
